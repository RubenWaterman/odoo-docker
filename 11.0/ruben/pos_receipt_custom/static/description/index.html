POS RECEIPT CUSTOM

Description

This module is used to customize default POS receipt. It improves your receipt when you print ticket from PosBox or
POS NETWORK PRINTER module. You have the possibility to print inline Designation, Quantity, Unit Price and Amount

Configuration

No configuration need. You install only the module from Apps menu.
